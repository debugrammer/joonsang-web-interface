# Joonsang.com
> joonsang.com by debugrammer

## Introduction
My simple portfolio website proudly based on Nuxt.js and Vuetify.js.

## Build Setup
``` bash
# Install dependency libraries
$ npm install

# Execute local Nodemon server
$ npm run dev

# Deploy
$ npm install
$ npm run build
$ pm2 start
```
